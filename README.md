# Frenchain Websockets Testbed

## Requirements
http-server or equivalent

## Depends
Jquery cdn, Web3 cdn

## Installation
```
npm i http-server -g
```

## Run it
```
http-server .
```

## Usage
Visit 127.0.0.1:8080/FREN_Web3_Websockets_Testbed.html in your browser!
